# 文件管理与播放器（M2）

## 目标

- 列表、标签、项目分类与搜索；内置播放器含波形、变速与书签。

## 范围

- UI：React + Ant Design；状态：Zustand；路由：tanstack router；样式：Tailwind。
- 播放器：`wavesurfer.js` + Web Audio；书签与备注写入数据库。

## 执行步骤

- 列表视图：排序/筛选；标签 CRUD；项目/文件夹归类。
- 搜索：文件名/标签/日期；为后续转写文本预留 FTS5。
- 播放器：变速不变调；波形渲染；书签时间点与文本备注；跳转联动。

## 验收标准

- 管理与搜索流畅；播放器稳定无卡顿；书签可新增/编辑/跳转。
