# 路线图与里程碑

## 目标

- 基于需求文档，按“MVP→迭代升级”路径分阶段交付可用版本。

## 里程碑

- M0 项目基线：工程可运行，安全基线启用，基础脚手架稳定。
- M1 设备连接与导出：自动识别设备、批量导出、导出后清理。
- M2 文件管理与播放器：列表/标签/项目分类，内置播放器与波形、书签。
- M3 备份与同步：本地/NAS 自动备份、增量策略与回收站机制。
- M4 基础编辑：剪辑、降噪与增强、格式转换；并发与重试策略。
- M5 AI 转写与标注：Whisper 集成、说话人分离、时间戳对齐与编辑。
- M6 协作分享：项目分享、协作编辑的最小方案（可选后端）。

## 执行顺序与验收

1. M0 完成：`pnpm qa`、`pnpm run build`通过；Electron 安全基线开启。
2. M1 完成：插入录音笔自动检测盘符；可选择目标路径一键导出并重命名；可选“导出后清空”。
3. M2 完成：文件列表可排序/搜索；标签与项目归类；播放器可变速、波形与书签。
4. M3 完成：本地/NAS定时备份；增量备份；删除进入回收站；可恢复。
5. M4 完成：可视化剪辑生成新片段；降噪/静音去除；格式转换；任务队列稳定。
6. M5 完成：本地离线转写；多说话人标注；文字与时间点双向跳转；文稿导出。
7. M6 完成：最小分享链路与权限；多人协作校对基本可用。

## 风险与缓解

- 设备识别差异：优先走可移动磁盘模式；品牌差异通过指引与白名单适配。
- 大文件性能：并发受控与分块处理；播放器走 `wavesurfer.js` 与 WebAudio。
- AI 模型体积：本地模型按需下载与缓存；提供关闭或延后配置。

## 产出

- `docs/plan/*` 分阶段计划与验收清单。
- 各阶段最小可运行演示与操作说明。
